# Web Copywriting & Design Guidelines

To deliver premium, senior-level frontend interfaces, strictly follow these copy and layout rules:

## 1. Eliminate Over-Explaining Paragraphs
- Avoid writing long paragraphs describing features. 
- Use short, concise copy (1-2 sentences maximum).
- Instead of paragraphs, list items inside visual layout blocks (e.g. grids, tables, or cards).

## 2. Emphasize Visual Hierarchy
- Use bold metrics, stats, badges, and inline tags.
- Present workflows using numbered bullet steppers, step timelines, or checklist icons (`CheckCircle`, `Check`).
- Use appropriate Lucide icons for every item to aid visual scanning.

## 3. Position Elements Intuitively
- Position peripheral elements like FAQs and Newsletter widgets at the bottom of pages, near the final Call to Action.
- Main conversion content and core proposition must remain at the top.
